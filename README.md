## 📌 About This Repository

This repository is a curated collection of **5 practical machine learning projects**, each built as a self-contained Google Colab notebook. Every project works with a real-world dataset, includes clear explanations in markdown cells, and is designed to run completely in the browser with no local setup required.

Whether you are just starting your ML journey or looking to sharpen your skills with hands-on practice, this lab has something for you.

## 📂 Projects

| # | Project | Description | Difficulty |
|---|---------|-------------|------------|
| 01 | [Student Performance Predictor](#) | Predict a student's final grade based on study hours, attendance, and past scores | ⭐⭐ |
| 02 | [Movie Genre Classifier](#) | Classify movies into genres using plot summary text and TF-IDF + Naive Bayes | ⭐⭐ |
| 03 | [Air Quality Index Forecaster](#) | Forecast AQI levels using historical pollution data and regression models | ⭐⭐⭐ |

## 🚀 How to Run (Google Colab)

1. Open [Google Colab](https://colab.research.google.com)
2. Click **File → Upload Notebook** and select any `.ipynb` file from the `notebooks/` folder
3. Download the required dataset (link provided inside each notebook)
4. Run the first cell — it will prompt you to upload the dataset file
5. Click **Runtime → Run All** to execute the full notebook top to bottom
6. Read the markdown explanations in each section to understand what the code does

> 💡 Tip: Use **Runtime → Change Runtime Type → GPU** for faster training in notebooks that use neural networks.
